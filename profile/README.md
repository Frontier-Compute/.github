# Frontier Compute

On-chain attestation infrastructure for Zcash. NSM1 structured memo protocol with BLAKE2b Merkle commitments anchored to mainnet.

## Public repositories

- [`nsm1`](https://github.com/Frontier-Compute/nsm1) - reference implementation. Payment receiver, scanner, anchor subsystem, verification endpoints. MIT.
- [`nsm1-verify`](https://github.com/Frontier-Compute/nsm1-verify) - standalone Merkle proof verification library. Rust + WASM. On [crates.io](https://crates.io/crates/nsm1-verify). MIT.
- [`nsm1-js`](https://github.com/Frontier-Compute/nsm1-js) - JavaScript/TypeScript verification SDK. WASM backend. 19 tests. MIT.
- [`nsm1-explorer`](https://github.com/Frontier-Compute/nsm1-explorer) - attestation browser. Dashboard, leaf search, proof viewer. MIT.
- [`nsm1-simulator`](https://github.com/Frontier-Compute/nsm1-simulator) - interactive 9-step lifecycle simulator with client-side Merkle tree. MIT.
